# 🔍 Sonar Mine vs Rock Prediction

## 📖 Project Overview

This is a machine learning project that predicts whether an object detected by sonar signals is a **mine** or a **rock**.
The model is trained using sonar frequency data and applies classification algorithms to make predictions.

---

## 🎯 Objective

The goal of this project is to build a classification model that can:

* Analyze sonar signal data
* Accurately classify objects as **Mine (M)** or **Rock (R)**

---

## 📊 Dataset

* The dataset used is the **Sonar Dataset**
* It contains multiple numerical features representing sonar signal returns
* Target labels:

  * `M` → Mine
  * `R` → Rock

---

## 🚀 How to Run the Project

1. Clone the repository:

```
git clone https://github.com/Naresh-1107/sonar-mine-prediction.git
```

2. Navigate to the project folder:

```
cd sonar-mine-prediction
```

3. Install required libraries:

```
pip install -r requirements.txt
```

4. Open the Jupyter Notebook:

```
jupyter notebook
```

5. Run all cells in the notebook
